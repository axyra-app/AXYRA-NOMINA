CÓMO EMPEZAR
============

1. Descargar serviceAccountKey.json (10 min)
   → Firebase Console → Configuración → Cuentas de servicio
   → Generar clave privada → Guardar en backend/

2. Instalar dependencias (5 min)
   → setup-dev.bat  (Windows)
   → .\setup-dev.ps1 (PowerShell)

3. Iniciar desarrollo (1 min)
   Terminal 1: cd backend && python main.py
   Terminal 2: cd frontend && npm run dev
   Navegador: http://localhost:5173

4. Verificar (2 min)
   → python verificar_sistema.py


ENDPOINTS DISPONIBLES
=====================

AUTENTICACIÓN:
  POST   /auth/signup           Registrarse
  POST   /auth/login            Iniciar sesión
  GET    /auth/me               Datos del usuario actual

EMPLEADOS:
  GET    /employees             Listar todos
  GET    /employees/{id}        Obtener uno
  POST   /employees             Crear
  PUT    /employees/{id}        Actualizar
  DELETE /employees/{id}        Eliminar

HORAS:
  GET    /hours                 Listar todas
  GET    /hours/{id}            Obtener una
  POST   /hours                 Registrar
  PUT    /hours/{id}            Actualizar
  DELETE /hours/{id}            Eliminar

NÓMINA:
  GET    /payroll               Historial completo
  GET    /payroll/{id}          Obtener una
  POST   /payroll/calculate     Calcular nómina
  GET    /payroll/batch         Lotes por período

CONFIGURACIÓN:
  GET    /configuration         Obtener config
  PUT    /configuration         Actualizar config

UTILITARIOS:
  GET    /health                Health check
  GET    /api/status            Estado del API
